# Pet project "TodoList"
Приложения для планирования задач, их учёта и формирования статистики.

                <li><h3 align="center">✔ Отображение задач поверх окон рабочего стола в компактном виде.</h3></li>

                <li><h3 align="center">✔ Экран "Календарь": со всеми задачами.</h3></li>
